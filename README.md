# bugTracker1
Web app for organizing bug report

Features:

    Dual Views: Switch between a List View and a Kanban Board View.

    Dashboard: Real-time statistics on open, closed, and high-priority issues.

    Ticket Management: Create new tickets, change status (Open -> In Progress -> Closed), and delete tickets.

    Search & Filter: Filter tickets by title or ID.

    Responsive Design: Works on desktop and mobile.
